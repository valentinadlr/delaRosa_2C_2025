\chapter{Visualización de número en Display BCD }
\hypertarget{index}{}\label{index}\index{Visualización de número en Display BCD@{Visualización de número en Display BCD}}
\hypertarget{index_genDesc}{}\doxysection{\texorpdfstring{General Description}{General Description}}\label{index_genDesc}
Este programa implementa la conversión de un número entero a formato BCD y lo muestra a través de un conjunto de pines GPIO del microcontrolador ESP32, que controlan tanto las líneas de datos como la selección de cada dígito del display. ~\newline
