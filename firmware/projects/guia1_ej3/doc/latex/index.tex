Este programa permite controlar el encendido de los LEDs mediante una estructura que define\+: ~\newline

\begin{DoxyItemize}
\item El número de LED a controlar. ~\newline

\item El modo de operación (ON, OFF, TOGGLE). ~\newline

\item La cantidad de ciclos de parpadeo. ~\newline

\item El período de cada ciclo de encendido/apagado. ~\newline
